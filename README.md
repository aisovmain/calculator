# 🧮 Калькулятор — Консольный Вычислитель


**Консольное приложение для выполнения арифметических операций с проверкой ввода.**

## ✅ Функциональность

| Операция | Описание | Пример |
|----------|----------|--------|
| `+` | Сложение | `5 + 3` |
| `-` | Вычитание | `10 - 4` |
| `*` | Умножение | `6 * 7` |
| `/` | Деление | `20 / 5` |


## 🚀 Запуск

# Скачай
https://github.com/aisovmain/calculator/releases/tag/calc

# Распакуй

# Пользуйся!
